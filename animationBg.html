Project TrackerX

A full stack application where after authentication, projects can be created and tickets submitted, documented, tracked and completed. Other features include: assigning User roles and assigning devs to separate projects.

  • Authorisation
  • Users/Roles

  • Project Tickets
  • Notifications
  • Ticket Update Chatroom
  • noSQL CRUD
